Write a program that maintains a collection of users where each user has a unique identifier. Manage the collection by adding new users with unique integers as their IDs and user names as Strings. If a user tries to add a user with an ID that already exists, your program should print 'ID exists'. Provide the functionality to also remove a user by their ID. The program should process a series of inputs representing the command ('add' or 'remove'), followed by the user ID, and for 'add' operations, the user name. The program ends when the input command is 'end'. After the final command, print out the remaining user IDs and names in the order they were added. Inputs consist of command ID pairs or command ID name triples, separated by spaces. Outputs should be the available user IDs and names after all operations, one per line, or 'ID exists' where appropriate.


Sample Input:
add 1 Alice
add 2 Bob
add 1 Charlie
remove 2
end

Sample Output:
ID exists
1 Alice


Sample Input:
add 3 Eve
remove 3
add 4 David
end

Sample Output:
4 David



Memory limit: 256 MB
Time limit: 8 seconds
