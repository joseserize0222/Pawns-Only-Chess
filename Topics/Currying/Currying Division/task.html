Implement a curried function named divide() that accepts two integers and returns a double representing the division of the first integer by the second. Demonstrate the usage of this function in Kotlin by dividing the call into two steps, each with one argument.

For example, using the implemented divide() function, the division of 10 by 2 should be as follows:

val quotient = divide(10)(2) // returns 5.0


Sample Input:
4 2

Sample Output:
2.0



Memory limit: 256 MB
Time limit: 8 seconds
