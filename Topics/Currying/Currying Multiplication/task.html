Write a function named multiply(), applying the principles of currying. The function should take two integer inputs and return their product as an integer output. This call should be conducted in two steps, each with one argument, according to the concept of currying.

For example, with the defined multiply() function, the multiplication of 5 and 3 should be executed as follows:

val product = multiply(5)(3) // returns 15


Sample Input:
1 2

Sample Output:
2



Memory limit: 256 MB
Time limit: 8 seconds
